# Parten Ranch Homeowner's Guide

**A searchable reference for every HOA rule, state law, and local ordinance that applies to our neighborhood — built in an hour with AI.**

**[View the guide](https://anfrasca.github.io/parten-ranch-guide/)**

---

## The Problem

I live in Parten Ranch, a new development in Dripping Springs, TX where the builder still runs the HOA. Neighbors in our community group were constantly frustrated: they'd submit architectural review applications to build a fence, add solar panels, or put in a pool, and get denied for reasons that seemed arbitrary. The "rules" cited in denial letters didn't appear in any of the governing documents on the HOA portal.

I experienced it myself when building our pool and outdoor kitchen. The process was opaque, the feedback was inconsistent, and there was no single place to look up what was actually required.

## What I Did

1. **Downloaded every governing document** from the HOA portal — CC&Rs, architectural guidelines, bylaws, plat restrictions. Most were scanned PDFs (images, not searchable text).

2. **Used Claude Code to extract and structure the content** — AI-powered OCR and conversion from scanned PDFs to clean, searchable markdown. This was the most tedious part; scanned documents with inconsistent formatting required iterative extraction and cleanup.

3. **Pulled in relevant Texas property code and local ordinances** — state laws that override HOA rules (like solar panel rights under the Texas Property Code), Dripping Springs municipal regulations, and Hays County requirements.

4. **Built a single searchable HTML reference** — organized by project type (pools, fences, pergolas, solar, landscaping, etc.) so a homeowner can look up exactly what the documents say, which rules are legally enforceable, and where state law provides protections the HOA can't override.

5. **Shared it with the neighborhood** — posted in the community Facebook group. The response was immediate: neighbors finally had a clear, authoritative reference they could cite when dealing with the HOA.

## Why This Matters (Beyond the Neighborhood)

This took about an hour from idea to live deployment. A year ago, it would have required hiring someone to manually transcribe scanned documents, a lawyer to cross-reference state law, and a developer to build the search interface. Instead, one person with AI tools turned a community frustration into a community resource in a single sitting.

That's the shift. Not "AI can write code" — but **AI lets someone who understands a problem deliver a complete solution at a speed that changes what's worth building.**

Not a lawyer. This isn't legal advice. The guide is a reference compiled from publicly available documents. If you're dealing with a specific HOA dispute, consult an attorney. If you spot an error, let me know.
